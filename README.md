# France visa — Telegram assistant

A small **Telegram** assistant for a visa workflow: it walks users through **France visa** basics, shares a **PDF guide**, and passes structured **contact requests** to the agent’s team. Submissions are stored locally and the team gets **notifications** in a private Telegram chat.

**Stack:** Python, Telegram Bot API, SQLite.

This repository is shared as a **delivery reference** for stakeholders (what was built and how it behaves), not as a turnkey setup guide. Hosting, tokens, and operational details stay with the maintainer.
