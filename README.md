# Telegram бот "Виза во Францию"

Бот для информирования пользователей о визе во Францию, предоставления гайда и сбора контактных данных для визового агента.

## Возможности

- ✅ Приветственное сообщение при `/start`
- ✅ Меню с двумя кнопками:
  - "Все о визе во Францию" - отправка PDF гайда
  - "Связаться с агентом" - сбор контактных данных
- ✅ Сохранение данных пользователей в базу данных (SQLite)
- ✅ Автоматическая отправка уведомлений о новых заявках в чат для заказчика и агента
- ✅ Проверка корректности введенных контактов (телефон или username)

## Установка и настройка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен бота
4. Настройте описание бота (Bot Description) через команду `/setdescription` в BotFather

### 3. Настройка конфигурации

1. Скопируйте файл `.env.example` в `.env`:

```bash
cp .env.example .env
```

2. Откройте `.env` и заполните необходимые параметры:

```env
BOT_TOKEN=ваш_токен_от_BotFather
NOTIFICATION_CHAT_ID=id_чата_для_уведомлений
PDF_FILE_PATH=guide.pdf
CHANNEL_LINK=https://t.me/your_channel
```

### 4. Как узнать ID чата для уведомлений

**Вариант 1:**

1. Создайте группу в Telegram
2. Добавьте в группу бота [@userinfobot](https://t.me/userinfobot)
3. Отправьте любое сообщение в группу
4. Бот покажет ID группы (будет отрицательное число, например: `-1001234567890`)

**Вариант 2:**

1. Добавьте бота в группу
2. Отправьте любое сообщение в группу
3. Перейдите по ссылке: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите в ответе `"chat":{"id":-1001234567890}` - это и есть ID чата

### 5. Подготовка PDF файла

Поместите файл с гайдом в корневую папку проекта и назовите его `guide.pdf` (или укажите другой путь в `.env`)

### 6. Запуск бота

```bash
python main.py
```

## Редактирование текстов

Все тексты сообщений можно редактировать в файле `config.py` без изменения основного кода:

- `WELCOME_MESSAGE` - приветственное сообщение при `/start`
- `AFTER_PDF_MESSAGE` - текст после отправки PDF
- `CONTACT_REQUEST_MESSAGE` - запрос контактных данных
- `CONTACT_RECEIVED_MESSAGE` - подтверждение получения контакта
- `NOTIFICATION_MESSAGE_TEMPLATE` - формат уведомления в чат

## База данных

Данные пользователей сохраняются в SQLite базу данных `bot_database.db`. Таблица `contacts` содержит:

- `user_id` - ID пользователя Telegram
- `username` - Имя пользователя
- `contact` - Контактные данные (телефон или username)
- `contact_type` - Тип контакта ('phone' или 'username')
- `created_at` - Дата и время создания записи

## Автозапуск бота (рекомендуется)

Для постоянной работы бота без необходимости держать терминал открытым используйте системный сервис macOS (launchd).

### Установка как системного сервиса

**Один раз выполните:**

```bash
./install_bot.sh
```

После этого бот будет:

- ✅ Автоматически запускаться при загрузке Mac
- ✅ Автоматически перезапускаться при сбоях
- ✅ Работать в фоне без терминала
- ✅ Не требовать вашего участия

### Управление ботом

```bash
# Запустить бота
launchctl start com.visabarbi.bot

# Остановить бота
launchctl stop com.visabarbi.bot

# Проверить статус
launchctl list | grep visabarbi

# Посмотреть логи
tail -f bot.log
```

### Удаление автозапуска

Если нужно убрать автозапуск:

```bash
./uninstall_bot.sh
```

## Альтернативные способы запуска

### Вариант 1: nohup (простой способ)

```bash
nohup python3 main.py > bot.log 2>&1 &
```

Бот будет работать в фоне, логи сохраняются в `bot.log`.  
Остановить: `pkill -f "python3 main.py"`

### Вариант 2: screen (удобно для управления)

```bash
# Создать сессию
screen -S visa_bot

# Запустить бота
python3 main.py

# Отключиться (не закрывать): Ctrl+A, затем D
# Вернуться к сессии: screen -r visa_bot
# Закрыть сессию: screen -X -S visa_bot quit
```

### Вариант 3: tmux (аналогично screen)

```bash
# Создать сессию
tmux new -s visa_bot

# Запустить бота
python3 main.py

# Отключиться: Ctrl+B, затем D
# Вернуться: tmux attach -t visa_bot
```
